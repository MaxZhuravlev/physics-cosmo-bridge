# Causal Typicality v3: Local MaxEnt Emergence Test

**Date**: 2026-02-16

**Question**: Does local MaxEnt emerge from causal invariance (CI) in a simplified toy model?

## Executive Summary

- **KL convergence (1st order MaxEnt)**: CI=71.1%, non-CI=26.7%
- **KL convergence (2nd order MaxEnt)**: CI=13.3%, non-CI=0.0%
- **CI lower KL than non-CI (1st order)**: 31.1% of configurations
- **CI lower KL than non-CI (2nd order)**: 75.6% of configurations

## Verdict

**INCONCLUSIVE**: Mixed signals, results depend on parameter regime.

## Model Description

**Simplified CI-like dynamics** (not actual Wolfram hypergraphs):
- State space: binary strings of length L
- CI rules: confluent (non-overlapping positions)
- Non-CI rules: non-confluent (overlapping positions, for control)

**Local observer**:
- Sees window of w consecutive bits (w << L)
- Does NOT know global state or global constraints
- Collects statistics over N evolution steps

**MaxEnt reference distributions**:
- 1st order: p ∝ exp(Σ λ_i x_i) matching observed means
- 2nd order: p ∝ exp(Σ h_i x_i + Σ J_ij x_i x_j) matching means + correlations (Ising)

**Measure**: KL(p_obs || p_MaxEnt)
