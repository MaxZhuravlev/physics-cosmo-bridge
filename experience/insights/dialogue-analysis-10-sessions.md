## Main Goal

## Key Findings

### 1. Цепочка Лавлока (КРИТИЧНО)

**Открытие.** CI → диффеоморфизмная симметрия (Горард 2020) → теорема Лавлока (1971) → единственность тензора Онзагера Ванчурина (Eq. 93).

Первая формальная связь двух программ, отвечающая на открытый вопрос самого Ванчурина: "can symmetries be derived from first principles?" (arXiv:2008.01540).

Никто ранее не применял теорему Лавлока для связи программ.

### 2. Квантовый сектор через Кирибеллу (КРИТИЧНО)

**Открытие.** 5/5 аксиом Кирибеллы выполняются в мультивее:
- Causality: CI → DAG → нет обратной причинности
- Perfect distinguishability: конфлюэнтное скалярное произведение G = A^T A (положительно определено по теореме)
- Ideal compression: rate-distortion, наблюдение = сжатие
- Local distinguishability: проверено исчерпывающим перебором 1134/1134 систем с null_dim = 0
- Purification: мультивей = compact structure (eta, epsilon)

G = A^T A — главный технически новый вклад.

### 3. Fisher = Riemann (ВАЖНО)

**Метод.** Обе метрики удовлетворяют единственным уравнениям гравитации → описывают одну физику → диффеоморфны. Следует из цепочки Лавлока без дополнительных допущений.

### 4. Дираковская структура M+M- (ВАЖНО)

**Открытие.** M+M- ~ alpha*M^2 обнаружена через ориентацию переходов по размеру каузального будущего (descendants). Ошибка 1.2-15% на слоевых матрицах.

Единственное предсказание, которого НЕТ ни у Вольфрама, ни у Ванчурина. Предварительно на toy models.

### 5. Стрела времени (ВАЖНО)

**Метод.** dL/dt = -g^{ij}(d_i L)(d_j L) <= 0 (g положительно определена) + каузальный граф ацикличен (из CI). Необратимость — структурная необходимость, не зависит от начальных условий.

### 6. d_eff ~ 1.6 — артефакт кодирования (ВАЖНО)

**Препятствие.** d_eff ~ 1.6 (participation ratio Фишера) объясняет K ~ N^1.6, но ТОЛЬКО для посимвольного кодирования. Hash-кодирование даёт d_eff ~ 10.7. Это артефакт представления, не фундаментальное свойство мультивея.

### 7. Конфлюэнтность != унитарность (ВАЖНО)

**Препятствие.** Отклонение 0.67-1.0. Закрывает ложный путь в КМ-секторе. Правильный путь — через аксиомы Кирибеллы.

### 8. Наблюдение = сжатие (СРЕДНЕ)

**Открытие.** Случайные проекции имеют выше MI (0.79 vs 0.54), но предсказывают в 22x хуже. Наблюдение — мера качества сжатия (rate-distortion).

### 9. CI-breaking test (КРИТИЧНО)

**Действие.** Wolfram с forced ordering даёт improvement +0.0% vs +4.3% для полного мультивея (p = 0.0215). Forced ordering сохраняет граф (устраняя confound), но нарушает CI.

Первый статистически значимый фальсифицируемый тест моста.

### 10. Оценка оригинальности (КРИТИЧНО)

**Препятствие.** ~60% работы — переоткрытие и систематизация известного:
- CI → ОТО (Горард 2020)
- NG (Амари 1998)
- Fisher → Эйнштейн (Ванчурин 2020)

Реально новое: связка через Лавлок, G = A^T A для LD, M+M- ~ alpha*M^2, d_eff = артефакт.

## Obstacles and Limitations

### Priority 1

- **Континуальный предел:** переход дискретная ковариантность → диффеоморфизмная инвариантность не доказан формально. Горард показал Ollivier-Ricci → Ricci, но полного доказательства нет. Единственное допущение в цепочке Лавлока.

- **Дирак на настоящих гиперграфах:** M+M- ~ alpha*M^2 проверен только на toy models (string rewriting). Требуется Wolfram Engine + SetReplace.

### Priority 2

- **Публикация без аффилиации.** Стратегия: заметка Ванчурину → arXiv preprint.

- **Персистентность наблюдателя:** второй вход цепочки Амари. Обосновано через Конант-Эшби (минимальное требование).

- **K ~ N^alpha механизм:** alpha ~ 0.76, корреляция с конфлюэнтностью r = -0.68, но механизм не формализован.
